As a normal user I want to enter a password to check its strength and security.

As a normal user I want to see specific feedback about my password to make it more secure.

As a normal user I want to compare to a list of known passwords to make sure it is not too common.

As a normal user I want to see the application's version to check that I have the right version installed.

As an administrtor I want to be able to see errors logged to the file system to fix potential problems in the future.
